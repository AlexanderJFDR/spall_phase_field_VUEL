# Spall Phase-Field Program

This program is developed using Abaqus VUEL subroutines to simulate spall fracture via a phase-field approach. The provided examples and material properties are based on the following reference:

Zhang, H., Peng, H., Pei, X. Y., Li, P., Tang, T. G., & Cai, L. C. (2021). 
A phase-field model for spall fracture. *Journal of Applied Physics*, 129(12).

## Directory Structure

- Code Files: Contains VUEL source codes and input files for 1D, 2D_A, and 2D_B cases.
- Free Surface Velocity Data: Simulation results for each case.
- Videos: Visualization of simulation results for each case.
